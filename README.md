# n8n Automations

This repository contains my automation workflows built using n8n.

## Included Workflows

- Facebook Lead Ads to Google Sheets Sync
- Webhook based Lead Processing
- CRM Automation Pipelines
- API Integrations

## Tools Used

- n8n
- Google Sheets
- Facebook Lead Ads API
- Webhooks
- HTTP Requests

## How to Import

1. Open n8n
2. Click Import
3. Upload JSON file
4. Activate workflow
